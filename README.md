# NOOR Collections 👗
An e-commerce web application for luxury evening wear built with PHP & sessions.

## 📌 Subject
server side scripting — jain university

## 🔧 Tech Stack
- PHP (Sessions, POST/GET handling)
- HTML5 / CSS3
- Bootstrap 4 / Font Awesome

## ✨ Features
- Product catalog with Add to Cart
- Session-based cart with quantity controls
- Remove items / Clear cart
- INR pricing with live cart total

## 🚀 How to Run
1. Install XAMPP
2. Place folder in `C:/xampp/htdocs/noor/`
3. Start Apache in XAMPP Control Panel
4. Open `http://localhost/noor/index.php`
